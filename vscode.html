VS code configs

1. Choose Tag pair (Alt + D)

( Chọn cặp thẻ div )

* Ấn 2 lần D để tiếp tục chọn nhiều div cha của phần tử đang chọn

2. Hotkey group tag (ctrl + shift + G)

( Cách để thêm thẻ div bao bên ngoài div có sẵn )

Setting -> Keyboard Shortcut -> search "wrap"

Ctrl + shift + G vào div cần bao bọc -> gõ div.tênclass -> enter

3. Remove tag (ctrl + shift + ;)

( Xóa 1 div )

Setting -> Keyboard Shortcut -> search "remove"

4. Go to matching pair (Ctrl + Shift + J)
5. Duplicate line code (Ctrl + Shift + D)
6. Prevent auto semicolon
  1. Press Control + Shift + P in VSCode.
  2. Search For Preferences: Open Settings (UI) .
  3. Again in the Searchbar, search for Semicolon .
  4. Choose CSS or your preferred language from where you want to remove auto semicolon.
  5. Untick Insert semicolon at the end of line when completing CSS properties .
